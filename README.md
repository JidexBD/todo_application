# Simple Todo/Notes App
##  simple todo app
       A simple productivity mobile app built with Flutter that helps users manage daily tasks and personal notes in one place.

    The app focuses on clarity and ease of use. You can add, edit, delete, and mark tasks as completed. Notes   work the same way, making it easy to capture ideas or reminders quickly.

## Features
   Create, edit, and delete todo tasks
   Mark tasks as completed
   Create, edit, and delete notes
   Separate sections for Todos and Notes
   Clean and simple user interface
## Tech Stack
     Flutter
     Dart
